# IC-9700 CI-V Web 控制器 — 使用说明

## 快速使用（独立 EXE，无需 Python）

1. 复制 `dist\ic9700-ctrl.exe` 到任意位置
2. **双击运行** `ic9700-ctrl.exe`
3. 浏览器自动打开控制页面 (http://127.0.0.1:8080)
4. 输入电台 IP 地址，点击「连接」
5. 按 `Ctrl+C` 或关闭命令行窗口退出

> 所有功能已打包在单个 exe 文件中，无需安装 Python 或任何依赖。

## 自行打包

```bash
cd "D:\IC9700 CIV CTRL"
build.bat

```

输出: `dist\ic9700-ctrl.exe`

## 连接电台

### 方式 1: LAN 连接（推荐）

1. 确保电台已连接到局域网（Wi-Fi 或有线）
2. 在页面顶部选择 **LAN**
3. 输入电台的 **IP 地址**（电台菜单 → Network → IP 地址）
4. 输入电台的 **用户名** 和 **密码**（电台菜单 → Network → 网络控制密码，默认通常为空）
5. 点击 **连接**
6. 连接成功后状态指示灯变绿，显示 "LAN已连接"

> 注意: 串口连接模式目前仍支持，但 LAN 连接是主要推荐方式。LAN 模式通过 UDP 端口 50001 (控制) 和 50002 (CI-V 数据) 与电台通信。

### 方式 2: 串口连接

1. 用 USB 线连接电台和电脑
2. 安装 Icom 串口驱动
3. 在页面选择 **串口**
4. 从下拉框选择对应的 COM 口
5. 波特率默认 115200
6. 点击 **连接**

## 界面操作

### 左侧导航

页面左侧是树形功能导航，点击可展开/折叠分类：

| 分类 | 内容 |
|------|------|
| 主显示 | 频率/模式显示、仪表盘 (S-Meter, PO, SWR, ALC 等) |
| 主控制 | 频率设置、模式选择、VFO 操作、记忆通道、扫描 |
| 发射控制 | Split/双工、RIT/XFC、RF 功率、语音记忆 |
| 接收控制 | AF/RF/SQL 滑块、PBT 滤波器、NB、NR、陷波器、AGC/前置放大 |
| 发射音频 | MIC 增益、压缩器、监听/VOX、Break-In |
| Function 设置 | Beep、TX延迟、Split/中继器、RTTY、语音、锁定、键盘等 |
| DV/DD 设置 | 数字语音、DD 模式设置 |
| Connectors 设置 | 外接前置放大、ACC/USB/LAN 接口配置 |
| Network 设置 | DHCP、IP、网络控制 |
| Display 设置 | LCD背光、显示字体、呼号显示、单位、语言 |
| Scope/Audio/KW/RTTY/Recorder/GPS 等 | 电台进阶功能设置 |
| 原始命令 | 手动发送 16 进制 CI-V 命令 |
| 通信日志 | 实时显示收发日志 |

### 主显示页面

连接成功后可实时查看:
- **频率**: 精确到 1Hz
- **模式**: LSB/USB/AM/CW/RTTY/FM/CW-R/RTTY-R/DV/DD
- **滤波器**: FIL1/FIL2/FIL3
- **收发状态**: RX (绿色) / TX (红色)
- **仪表条**: S-Meter, PO, SWR, ALC, COMP, Vd, Id（自动更新）

### 功能控制方式

| 控件类型 | 用途 | 操作方式 |
|----------|------|----------|
| **开关按钮** | 开关类功能 (ON/OFF) | 点击切换 |
| **滑块** | 连续值 (0-255) | 拖动调整，松开即发送 |
| **下拉选择** | 多选项功能 | 选择后自动发送 |
| **设置频率** | 输入频率 (Hz) | 输入数值后点击按钮 |
| **模式按钮** | 切换模式 | 点击即切换 |

### 滑块使用

- 拖动滑块调整参数值
- 松开鼠标时自动发送到电台
- 当前值显示在滑块上方
- 大多数电平参数范围为 0-255

### 开关按钮使用

- 二值开关 (ON/OFF): 点击切换状态
- 多值选择: 显示为下拉框样式
- 状态通过轮询实时同步

### 原始命令

在 **原始命令** 页面可以手动输入 CI-V 十六进制命令：

例: 读取频率
```
FEFEA2E003FD
```

例: 设置频率为 146.520 MHz
```
FEFEA2E005000052146000FD
```

## 常见问题

### 连接失败

1. 检查电台 IP 地址是否正确
2. 检查电脑与电台是否在同一局域网
3. 确认电台的 **网络控制** 功能已开启（菜单 → Network → 网络控制 → ON）
4. 确认用户名密码（如未设置则留空）
5. 防火墙可能拦截 UDP 端口 50001/50002

### 连接成功但无数据显示

1. 检查通信日志是否有 TX 但没有 RX 响应
2. 确认 CI-V 地址设置正确（IC-9700 默认 0xA2）
3. 确认电台的 CI-V Transceive 功能（菜单 → Connectors → CI-V → CI-V 收发）
4. 查看 `app.log` 日志文件获取详细诊断

### 频谱数据

本系统已自动过滤频谱/瀑布数据（CI-V 命令 0x27），不会影响正常使用。如果需要频谱功能，请使用 wfview 等专业软件。

### 多用户

多个浏览器标签页可以同时连接，所有页面会实时同步显示。

## 日志

运行日志保存在 `app.log` 文件中，包含:
- LAN 连接过程 (发现、认证、握手)
- CI-V 收发的命令帧 (十六进制)
- 频谱帧过滤统计
- 连接超时等异常信息
