## 连接模式

### 连接方式 1：串口（USB/RS-232）
- 传输层: `civ.py` → `CIVSerial` (PySerial)
- 默认波特率: 115200
- CI-V 设备地址: 0xA2 (电台), 0xE0 (控制器)
- 数据帧: `FE FE [To] [From] [Cmd] [Data] FD`

### 连接方式 2：LAN（Wi-Fi/以太网）
- 传输层: `lan.py` → `LanCIVTransport`
- 控制端口: UDP 50001（认证、保活）
- CI-V 数据端口: UDP 50002（CI-V 命令隧道）
- 认证方式: Icom 私有密码编码算法（与 wfview 兼容）
- CI-V 数据封装在 UDP DATA 包内 (C1 + datalen + sendseq + CI-V frame)

### LAN 连接流程
```
1. 连接控制端口 (50001) → discover → IAmHere → ready → IAmReady
2. 发送 Login 包 (加密凭证) → 接收 Token
3. 发送 ConnInfo (CI-V 端口信息, 无音频/频谱) → 接收 Status
4. Token ACK
5. 连接 CI-V 端口 (50002) → discover → ready
6. 启动 CI-V 读循环 + 控制端口保活 (Ping/Idle)
```

## CI-V 协议细节

### 帧格式
```
FE FE [ToAddr] [FromAddr] [Cmd] [SubCmd?] [Data...] FD
```

| 字段 | 长度 | 示例 | 说明 |
|------|------|------|------|
| Preamble | 2B | FE FE | 帧同步头 |
| To Address | 1B | A2 | 目标地址 (0xA2=IC-9700, 0xE0=PC, 0x00=广播) |
| From Address | 1B | E0 | 源地址 |
| Command | 1B | 05 | 命令码 |
| Sub-command | 0-2B | 02 | 子命令（可选） |
| Data | 0-NB | BCD | 数据（BCD 编码） |
| End Code | 1B | FD | 帧结束 |

### 频率 BCD 编码
5 字节 little-endian 顺序：
- Byte 0: 10Hz, 1Hz
- Byte 1: 1kHz, 100Hz
- Byte 2: 100kHz, 10kHz
- Byte 3: 10MHz, 1MHz
- Byte 4: 1GHz, 100MHz

例: 146.520 MHz → `00 52 14 60 00`

### 工作模式
| 代码 | 模式 | 代码 | 模式 |
|------|------|------|------|
| 0x00 | LSB | 0x07 | CW-R |
| 0x01 | USB | 0x08 | RTTY-R |
| 0x02 | AM | 0x17 | DV |
| 0x03 | CW | 0x22 | DD |
| 0x04 | RTTY | 0x05 | FM |

## CI-V 命令参考

### 0x00 / 0x03 — 频率读写
- `FE FE A2 E0 03 FD` — 读取 VFO 频率
- `FE FE A2 E0 05 [5B BCD] FD` — 设置 VFO 频率
- 响应: `FE FE E0 A2 00 [5B BCD] FD` (自动报告) / `FE FE E0 A2 03 [5B BCD] FD` (查询响应)

### 0x04 / 0x06 — 模式读写
- `FE FE A2 E0 04 FD` — 读取当前模式
- `FE FE A2 E0 06 [Mode] [Filter] FD` — 设置模式
- 响应: `FE FE E0 A2 01 [Mode] [Filter] FD`

### 0x07 — VFO 选择
- 0x00: VFO A, 0x01: VFO B
- 0xA0: VFO A=B, 0xB0: VFO A↔B
- 0xD0: 选择主频段, 0xD1: 选择副频段

### 0x08-0x0B — 记忆通道
- `08 [H] [L]`: 选择记忆通道 (0x0000-0xFFFF)
- `09`: 记忆写入, `0A`: 复制 VFO→记忆, `0B`: 清除记忆

### 0x0E — 扫描控制
- 0x00: 取消扫描, 0x01: 编程记忆扫描, 0x02: 编程扫描
- 0x03: ΔF 扫描, 0x12: 精细编程, 0x13: 精细 ΔF
- 0x22: 记忆扫描, 0x23: 选择记忆扫描, 0x24: 模式选择扫描
- 0xA0+A: 设置扫描跨距, 0xD0/D3: 扫描恢复

### 0x0F — Split / 双工
- `0F 00`: 单工/关, `0F 01`: Split 开
- 0x10: 单工, 0x11: DUP-, 0x12: DUP+, 0x13: RPS

### 0x10 — 调谐步进
- 0x00: 10Hz/1Hz, 0x01: 100Hz, 0x02: 500Hz
- 0x03: 1kHz ... 0x0B: 100kHz

### 0x11 — 衰减器
- 0x00: OFF, 0x10: 10dB

### 0x14 — 电平控制 (3 字节数据: SubCmd + 2B Value)
| SubCmd | 功能 | 范围 |
|--------|------|------|
| 0x01 | AF 音量 | 0-255 |
| 0x02 | RF 增益 | 0-255 |
| 0x03 | 静噪 SQL | 0-255 |
| 0x06 | NR 电平 | 0-255 |
| 0x07 | PBT1 (CCW↔CW) | 0-255 |
| 0x08 | PBT2 (CCW↔CW) | 0-255 |
| 0x09 | CW 音调 | 300-900Hz |
| 0x0A | RF 功率 | 0-255 |
| 0x0B | MIC 增益 | 0-255 |
| 0x0C | 键速 | 6-48 WPM |
| 0x0D | 陷波器位置 | 0-255 |
| 0x0E | COMP 电平 | 0-255 |
| 0x0F | Break-In 延迟 | 0-255 |
| 0x12 | NB 电平 | 0-255 |
| 0x15 | 监听电平 | 0-255 |
| 0x16 | VOX 增益 | 0-255 |
| 0x17 | Anti-VOX | 0-255 |
| 0x19 | 背光亮度 | 0-255 |

### 0x15 — 仪表读取 (2 字节数据: SubCmd)
| SubCmd | 仪表 | 范围 |
|--------|------|------|
| 0x01 | 静噪状态 | 0/1 |
| 0x02 | S-Meter | 0-255 (0=S0, 120=S9, 180=S9+40) |
| 0x11 | PO (输出功率) | 0-255 (0-100%) |
| 0x12 | SWR | 0-120 (1.0-∞) |
| 0x13 | ALC | 0-120 |
| 0x14 | COMP | 0-255 (0-30dB) |
| 0x15 | Vd (电压) | 0-255 (0-17V) |
| 0x16 | Id (电流) | 0-255 (0-21A) |

### 0x16 — 功能开关 (2 字节数据: SubCmd + Value)
| SubCmd | 功能 | 取值 |
|--------|------|------|
| 0x02 | 前置放大 | 0=OFF, 1=P.AMP, 2=EXT, 3=BOTH |
| 0x12 | AGC 时间 | 1=FAST, 2=MID, 3=SLOW |
| 0x22 | NB (噪声消除) | 0/1 |
| 0x40 | NR (降噪) | 0/1 |
| 0x41 | 自动陷波 | 0/1 |
| 0x42 | 中继音 (Repeater Tone) | 0/1 |
| 0x43 | 音静噪 (Tone Squelch) | 0/1 |
| 0x44 | 语音压缩器 | 0/1 |
| 0x45 | 监听 MONI | 0/1 |
| 0x46 | VOX | 0/1 |
| 0x47 | BK-IN | 0=OFF, 1=Semi, 2=Full |
| 0x48 | 手动陷波 | 0/1 |
| 0x4A | AFC | 0/1 |
| 0x4B | DTCS | 0/1 |
| 0x4F | 双峰值滤波器 | 0/1 |
| 0x50 | 拨号锁定 | 0/1 |
| 0x56 | DSP IF 滤波器 | 0=SHARP, 1=SOFT |
| 0x57 | 陷波宽度 | 0=WIDE, 1=MID, 2=NAR |
| 0x58 | SSB 发射带宽 | 0=WIDE, 1=MID, 2=NAR |
| 0x59 | 副频段 | 0/1 |
| 0x5A | 卫星模式 | 0/1 |
| 0x5B | DSQL/CSQL | 0=OFF, 1=DSQL, 2=CSQL |
| 0x65 | IP Plus | 0/1 |

### 0x18 — 电源控制
- `18 00`: 关机
- `18 01`: 开机（需 5 字节 FE 前导）

### 0x19 — ID 读取
- `19 00`: 读取电台型号

### 0x1A — 扩展设置 (SubCmd 05 + Item + Data)
大量电台配置项，包括: Beep 设置、TX 延迟、Split 设置、RTTY 设置、语音设置、键盘设置、DV/DD 设置、连接器设置 (ACC/USB/LAN)、网络设置、显示设置、SCOPE 设置、音频 SCOPE、KEYER/CW 设置、录音设置、GPS 设置、DTMF 设置等。

### 0x1B — 音调/ DTCS 读取
- `1B 00`: 读取中继音频率
- `1B 01`: 读取 TSQL 频率
- `1B 02`: 读取 DTCS 码

### 0x1C — 收发状态
- `1C 00`: 读取 TX/RX 状态 (响应: 0=RX, 1=TX)
- `1C 02 [0/1]`: 读取/设置 XFC

### 0x21 — RIT
- `21 01 [0/1]`: 设置 RIT 开/关
- `21 [BCD] [0/1]`: 设置 RIT 频率和方向 (0=+, 1=-)

### 0x24 — TX 功率设置
- `24 00 [0/1]`: 设置 TX 输出

### 0x27 — 频谱/瀑布数据
- 由电台自动发送（本系统已过滤掉）
- 数据格式: 频谱像素值数组 (0x00-0xFF)

### 0x28 — 语音发射
- `28 [Ch]`: 播放语音记忆通道

## 轮询机制

后台线程每 150ms 轮流发送以下查询命令（共 8 个命令，完整周期 ~1.2 秒）:

| 命令 | 用途 |
|------|------|
| 0x03 | 读取主频率 |
| 0x04 | 读取模式/滤波器 |
| 0x15 0x02 | S-Meter |
| 0x15 0x11 | PO 功率 |
| 0x15 0x12 | SWR |
| 0x15 0x15 | Vd 电压 |
| 0x15 0x16 | Id 电流 |
| 0x1C 0x00 | TX/RX 状态 |

## 频谱过滤说明

IC-9700 在 LAN 连接后会自动发送频谱/瀑布数据（CI-V 命令 0x27），每帧约 500 字节，频率约 30Hz。系统在 `_civ_read_loop` 中过滤掉这些帧，不记录日志、不转发到前端，保证正常运行不受影响。
